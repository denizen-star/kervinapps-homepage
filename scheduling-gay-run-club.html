Scheduling Gay Run Club

Community event management for LGBTQ+ running club

TypeScript Planned

Overview

Scheduling Gay Run Club is a community event management platform designed to coordinate and manage running club events for the LGBTQ+ community. The app provides tools for organizing runs, managing attendance, and building a supportive community.

With built-in safety features and inclusive design, the platform helps create a welcoming space for LGBTQ+ runners of all levels to connect, stay active, and build lasting friendships.

Key Features

Event Scheduling

Easy-to-use interface for organizing regular runs and special events

Community Building

Foster connections among members with social features

Safety Tools

Built-in safety features for community protection

Attendance Tracking

Monitor participation and engagement metrics
